30 //
31 // Class description:
32 //
33 // Data structure bringing together a magnetic track's state.
34 // (position, momentum direction & modulus, energy, spin, ... )
35 // Uses/abilities:
36 // - does not maintain any relationship between its data (eg energy/momentum).
37 // - for use in Runge-Kutta solver (in passing it the values right now).
