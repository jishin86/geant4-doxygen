36 // This is a variant of the FTFP_BERT physics list, whereby CRMC is used
37 // for modeling final-state pion- , kaon- , proton- , neutron- and
38 // ion-nuclear inelastic interactions at very high energies (above a
39 // threshold defined in HadronPhysicsCRMC_FTFP_BERT and IonCRMCPhysics).
40 // For the remaining inelastic interactions (i.e. hyperon- , antihyperon- ,
41 // antinucleon- and light anti-ion-nuclear interactions, as well as
42 // for all elastic final-state interactions, and for all elastic and
43 // inelastic hadronic cross sections, the usual Geant4 approach, as in
44 // FTFP_BERT is used.
