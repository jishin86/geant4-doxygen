31 // Class Description:
32 //
33 // A singleton class for computation and storage of the tolerance values
34 // used by the geometry modeler for precision on boundaries.
35 // The Cartesian tolerance defines the thickness of the surface of the
36 // geometrical shapes for their inner funtions and for the tracking. The
37 // tolerance must be greater than the largest mathematical error from the
38 // shape distance calculation functions. The tolerance is centred on the
39 // surface, e.g. the Inside() method of a solid uses a tolerance dx +/- kTol/2.
40 // The Cartesian tolerance can either be set to a fixed value (1E-9 mm)
41 // or to a value computed on the basis of the maximum extent of the
42 // world volume assigned through the G4GeometryManager at the beginning
43 // of the application -before- any geometrical object is created.
