32 // G4EllipticalCone
33 //
34 // Class description:
35 //
36 // G4EllipticalCone is a full cone with elliptical base which can be cut in Z.
37 //
38 // Member Data:
39 //
40 // xSemiAxis semi-axis, x, without dimentions
41 // ySemiAxis semi-axis, y, without dimentions
42 // zheight height, z
43 // zTopCut upper cut plane level, z
44 //
45 // The height in Z corresponds to where the elliptical cone hits the
46 // Z-axis if it had no Z cut. Also the cone is centered at zero having a
47 // base at zTopCut and another at -zTopCut. The semi-major axes at the Z=0
48 // plane are given by xSemiAxis*zheight and ySemiAxis*zheight so that the
49 // curved surface of our cone satisfies the equation:
50 //
51 // ***************************************************************************
52 // * *
53 // * (x/xSemiAxis)^2 + (y/ySemiAxis)^2 = (zheight - z)^2 *
54 // * *
55 // ***************************************************************************
56 //
57 // In case you want to construct G4EllipticalCone from:
58 // 1. halflength in Z = zTopCut
59 // 2. Dx and Dy = halflength of ellipse axis at z = -zTopCut
60 // 3. dx and dy = halflength of ellipse axis at z = zTopCut
61 // ! Attention : dx/dy=Dx/Dy
62 //
63 // You need to find xSemiAxis,ySemiAxis and zheight:
64 //
65 // xSemiAxis = (Dx-dx)/(2*zTopCut)
66 // ySemiAxis = (Dy-dy)/(2*zTopCut)
67 // zheight = (Dx+dx)/(2*xSemiAxis)
68

70 // Dionysios Anninos, 8.9.2005
71 //
72 // Revisions:
73 // Lukas Lindroos, Tatiana Nikitina, 20.08.2007
74 // Evgueni Tcherniaev, 20.07.2017
75 //
