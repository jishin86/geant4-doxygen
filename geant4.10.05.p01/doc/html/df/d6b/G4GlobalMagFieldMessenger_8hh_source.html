27 // class G4GlobalMagFieldMessenger
28 //
29 // Class description:
30 //
31 // Global uniform magnetic field messenger class.
32 //
33 // It defines UI commands:
34 // - /globalField/setValue vx vy vz unit
35 // - /globalField/verbose level
36 //
37 // It creates/deletes the global uniform magnetic field and
38 // activates/inactivates it according to the set field value.
39 // The field value can be changed either interactively via
40 // the UI command or via SetFieldValue() function.
