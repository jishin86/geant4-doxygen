27 // class G4PathFinder
28 //
29 // Class description:
30 //
31 // This class directs the lock-stepped propagation of a track in the
32 // 'mass' and other parallel geometries. It ensures that tracking
33 // in a magnetic field sees these parallel geometries at each trial step,
34 // and that the earliest boundary limits the step.
35 //
36 // For the movement in field, it relies on the class G4PropagatorInField
37 //
38 // History:
39 // -------
40 // 7.10.05 John Apostolakis, Draft design
41 // 26.04.06 John Apostolakis, Revised design and first implementation
42 // ---------------------------------------------------------------------------
