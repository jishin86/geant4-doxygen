32 // G4Sphere
33 //
34 // Class description:
35 //
36 // A G4Sphere is, in the general case, a section of a spherical shell,
37 // between specified phi and theta angles
38 //
39 // The phi and theta segments are described by a starting angle,
40 // and the +ve delta angle for the shape.
41 // If the delta angle is >=2*pi, or >=pi the shape is treated as
42 // continuous in phi or theta respectively.
43 //
44 // Theta must lie between 0-pi (incl).
45 //
46 // Member Data:
47 //
48 // fRmin inner radius
49 // fRmax outer radius
50 //
51 // fSPhi starting angle of the segment in radians
52 // fDPhi delta angle of the segment in radians
53 //
54 // fSTheta starting angle of the segment in radians
55 // fDTheta delta angle of the segment in radians
56 //
57 //
58 // Note:
59 // Internally fSPhi & fDPhi are adjusted so that fDPhi<=2PI,
60 // and fDPhi+fSPhi<=2PI. This enables simpler comparisons to be
61 // made with (say) Phi of a point.
62

64 // 28.3.94 P.Kent: old C++ code converted to tolerant geometry
65 // 17.9.96 V.Grichine: final modifications to commit
66 // --------------------------------------------------------------------
