33 // G4Tubs
34 //
35 // Class description:
36 //
37 // A tube or tube segment with curved sides parallel to
38 // the z-axis. The tube has a specified half-length along
39 // the z-axis, about which it is centered, and a given
40 // minimum and maximum radius. A minimum radius of 0
41 // corresponds to filled tube /cylinder. The tube segment is
42 // specified by starting and delta angles for phi, with 0
43 // being the +x axis, PI/2 the +y axis.
44 // A delta angle of 2PI signifies a complete, unsegmented
45 // tube/cylinder.
46 //
47 // Member Data:
48 //
49 // fRMin Inner radius
50 // fRMax Outer radius
51 // fDz half length in z
52 //
53 // fSPhi The starting phi angle in radians,
54 // adjusted such that fSPhi+fDPhi<=2PI, fSPhi>-2PI
55 //
56 // fDPhi Delta angle of the segment.
57 //
58 // fPhiFullTube Boolean variable used for indicate the Phi Section
59
