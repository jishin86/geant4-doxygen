33 // G4GenericTrap
34 //
35 // Class description:
36 //
37 // G4GenericTrap is a solid which represents an arbitrary trapezoid with
38 // up to 8 vertices standing on two parallel planes perpendicular to Z axis.
39 //
40 // Parameters in the constructor:
41 // - name - solid name
42 // - halfZ - the solid half length in Z
43 // - vertices - the (x,y) coordinates of vertices:
44 // o first four points: vertices[i], i<4
45 // are the vertices sitting on the -halfZ plane;
46 // o last four points: vertices[i], i>=4
47 // are the vertices sitting on the +halfZ plane.
48 //
49 // The order of defining the vertices of the solid is the following:
50 // - point 0 is connected with points 1,3,4
51 // - point 1 is connected with points 0,2,5
52 // - point 2 is connected with points 1,3,6
53 // - point 3 is connected with points 0,2,7
54 // - point 4 is connected with points 0,5,7
55 // - point 5 is connected with points 1,4,6
56 // - point 6 is connected with points 2,5,7
57 // - point 7 is connected with points 3,4,6
58 // Points can be identical in order to create shapes with less than
59 // 8 vertices.
