28 // class G4LogicalVolumeStore
29 //
30 // Class description:
31 //
32 // Container for all LogicalVolumes, with functionality derived from
33 // std::vector<T>. The class is a `singleton', in that only
34 // one can exist, and access is provided via the static function
35 // G4LogicalVolumeStore::GetInstance()
36 //
37 // All LogicalVolumes should be registered with G4LogicalVolumeStore,
38 // and removed on their destruction. Intended principally for UI browser.
39 // The underlying container initially has a capacity of 100.
40 //
41 // If much additional functionality is added, should consider containment
42 // instead of inheritance for std::vector<T>.
43 //
44 // Member data:
