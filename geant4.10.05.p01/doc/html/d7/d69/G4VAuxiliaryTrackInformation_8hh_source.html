31 // G4VAuxiliaryTrackInformation
32 //
33 // Class Description:
34 //
35 // Abstract class which the user can derive his/her own concrete
36 // class for toring user's information associating with a G4Track
37 // class object.
38 //
39 // It is user's responsibility
40 // 1) Construct a concrete class object and set the pointer to
41 // proper G4Track object
42 // 2) Concrete class derived from this class is expected to use G4Allocator
43 // for memory management or something equivarent for performance reason
44 //
45 // To set a pointer of a concrete class object to G4Track in
46 // G4AuxiliaryTrackingInformation concrete implementation, given the G4Track
47 // object is available only by "pointer to const", SetAuxiliaryTrackInformation()
48 // method of G4TrackingManager is available.
49 //
50 // The concrete class object is deleted by Geant4 kernel when
51 // associated G4Track object is deleted.
