38 // This is a primitive scorer class for scoring Surface Flux.
39 // Current version assumes only for G4Box shape, and the surface
40 // is defined at the -Z plane of the box.
41 // The surface flux is given in the unit of area.
42 // e.g. sum of 1/cos(T)/mm2, where T is a incident angle of the
43 // track on the surface.
44 //
45 //
46 // Surface is defined at the -Z surface.
47 // Direction -Z +Z
48 // 0 IN || OUT ->|<- | fFlux_InOut
49 // 1 IN ->| | fFlux_In
50 // 2 OUT |<- | fFlux_Out
51 //
52 // Created: 2005-11-14 Tsukasa ASO, Akinori Kimura.
53 //
54 // 18-Nov-2005 T.Aso, To use always positive value for anglefactor.
55 // Bug fix. Area definition.
56 // 29-Mar-2007 T.Aso, Bug fix for momentum direction at outgoing flux.
57 // 2010-07-22 Introduce Unit specification.
58 // 2010-07-22 Add weighted and divideByArea options
