30 // G4RichTrajectory.hh
31 //
32 // class description:
33 // This class extends G4Trajectory, which includes the following:
34 // 1) List of trajectory points which compose the trajectory,
35 // 2) static information of particle which generated the
36 // trajectory,
37 // 3) trackID and parent particle ID of the trajectory.
38 // The extended information, only publicly accessible through AttValues,
39 // includes:
40 // 4) physical volume and next physical volume;
41 // 5) creator process;
42 // ...and much more.
43 //

45 // Questions and comments on G4Trajectory should be sent to
46 // Katsuya Amako (e-mail: Katsuya.Amako@kek.jp)
