36 // (Description)
37 // This is a primitive scorer class for scoring cell flux.
38 // The Cell Flux is defined by a track length divided by a geometry
39 // volume, where only tracks passing through the geometry are taken
40 // into account. e.g. the unit of Cell Flux is mm/mm3.
41 //
42 // If you want to score all tracks in the geometry volume,
43 // please use G4PSCellFlux.
