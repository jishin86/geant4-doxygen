43 // Class Description
44 // This is a base class for hadronic cross section data sets. Users may
45 // derive specialized cross section classes and register them with the
46 // appropriate process, or use provided data sets.
47 //
48 // Each cross section should have unique name
49 // Minimal and maximal energy for the cross section will be used in run
50 // time before IsApplicable method is called
51 //
52 // Both the name and the energy interval will be used for documentation
53 //
54 // Class Description - End
