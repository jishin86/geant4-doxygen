31 // Class description:
32 //
33 // Decides which intermediate points on a curved trajectory merit
34 // being stored. Defines the compromise between accuracy of
35 // representation of the curved trajectory and memory use.
36 //
37 // Derived classes should implement the filtering algorithm in the
38 // method TakeIntermediatePoint().
39 //
40 // IMPORTANT: This class heap allocates vectors of auxiliary points,
41 // which it does not delete. The vectors must find their way to a
42 // subclass of G4VTrajectoryPoint, which must take responsibility for
43 // deleting them.
