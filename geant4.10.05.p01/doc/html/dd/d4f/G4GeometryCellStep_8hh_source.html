29 // Class G4GeometryCellStep
30 //
31 // Class description:
32 //
33 // This class serves to address the "cell" a track previously
34 // touched and a "cell" a track is currently in. It is used
35 // for scoring and importance sampling in the "mass" geometry as well
36 // as in a "parallel" geometry.
37 // The "cell" information is available with the GetPreGeometryCell and
38 // the GetPostGeometryCell functions.
39 // The GetCrossBoundary finction returns true in case the step
40 // crosses a boundary in the geometry this G4GeometryCellStep
41 // refers to.
42

44 // ----------------------------------------------------------------------
