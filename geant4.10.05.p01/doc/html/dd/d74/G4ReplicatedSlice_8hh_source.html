28 // class G4ReplicatedSlice
29 //
30 // Class description:
31 //
32 // G4ReplicatedSlice represents many touchable detector elements differing
33 // only in their positioning. The elements' positions are calculated by means
34 // of a simple linear formula.
35 //
36 // Division may occur along:
37 //
38 // o Cartesian axes (kXAxis,kYAxis,kZAxis)
39 //
40 // The divisions, of specified width have coordinates of
41 // form (-width*(nReplicas-1)*0.5+n*width,0,0) where n=0.. nReplicas-1
42 // for the case of kXAxis, and are unrotated.
43 //
44 // o Radial axis (cylindrical polar) (kRho)
45 //
46 // The divisions are cons/tubs sections, centred on the origin
47 // and are unrotated.
48 // They have radii of width*n+offset to width*(n+1)+offset
49 // where n=0..nReplicas-1
50 //
51 // o Phi axis (cylindrical polar) (kPhi)
52 // The divisions are `phi sections' or wedges, and of cons/tubs form
53 // They have phi of offset+n*width to offset+(n+1)*width where
54 // n=0..nReplicas-1
