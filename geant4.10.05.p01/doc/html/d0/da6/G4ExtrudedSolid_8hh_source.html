33 // G4ExtrudedSolid
34 //
35 // Class description:
36 //
37 // G4ExtrudedSolid is a solid which represents the extrusion of an arbitrary
38 // polygon with fixed outline in the defined Z sections.
39 // The z-sides of the solid are the scaled versions of the same polygon.
40 // The solid is implemented as a specification of G4TessellatedSolid.
41 //
42 // Parameters in the constructor:
43 // const G4String& pName - solid name
44 // std::vector<G4TwoVector> polygon - the vertices of the outlined polygon
45 // defined in clockwise or anti-clockwise order
46 // std::vector<ZSection> - the z-sections defined by
47 // z position, offset and scale
48 // in increasing z-position order
49 //
50 // Parameters in the special constructor (for solid with 2 z-sections:
51 // G4double halfZ - the solid half length in Z
52 // G4TwoVector off1 - offset of the side in -halfZ
53 // G4double scale1 - scale of the side in -halfZ
54 // G4TwoVector off2 - offset of the side in +halfZ
55 // G4double scale2 - scale of the side in -halfZ
56

58 // Ivana Hrivnacova, IPN Orsay
59 //
