29 // class G4ReflectionFactory
30 //
31 // Class description:
32 //
33 // Class providing functions for volumes placements with a general
34 // transfomation that can contain reflection.
35 // Reflection is then applied to a solid: a new G4ReflectedSolid
36 // instance is created and is placed with a transformation containing
37 // pure rotation and translation only.
38 // The pair of constituent and reflected logical volumes is
39 // considered as a generalized logical volume that is addressed
40 // by user specifying the constituent logical volume.
41 //
42 // Decomposition of a general transformation that can include reflection
43 // in a "reflection-free" transformation:
44 //
45 // x(inM') = TG*x(inM) TG - general transformation
46 // = T*(R*x(inM)) T - "reflection-free" transformation
47 // = T* x(inReflM)
48 //
49 // Daughters transformation:
50 // When a volume V containing daughter D with transformation TD
51 // is placed in mother M with a general tranformation TGV,
52 // the TGV is decomposed. New reflected volume ReflV containing
53 // a new daughter ReflD with reflected transformation ReflTD is created:
54 //
55 // x(inV) = TD * x(inD);
56 // x(inM) = TGV * x(inV)
57 // = TV * R * x(inV)
58 // = TV * R * TD * x(inD)
59 // = TV * R*TD*R-1 * R*x(inD)
60 // = TV * ReflTD * x(inReflD)
