32 // G4Torus
33 //
34 // Class description:
35 //
36 // A torus or torus segment with curved sides parallel to the z-axis.
37 // The torus has a specified swept radius about which it is centered,
38 // and a given minimum and maximum radius. A minimum radius of 0
39 // signifies a filled torus.
40 // The torus segment is specified by starting and delta angles for phi,
41 // with 0 being the +x axis, PI/2 the +y axis. A delta angle of 2PI
42 // signifies a complete, unsegmented torus/cylindr.
43 //
44 // Member functions:
45 //
46 // As inherited from G4CSGSolid+
47 //
48 // G4Torus(const G4String &pName
49 // G4double pRmin
50 // G4double pRmax
51 // G4double pRtor
52 // G4double pSPhi
53 // G4double pDPhi )
54 //
55 // - Construct a torus with the given name and dimensions.
56 // The angles are provided is radians. pRtor >= pRmax
57 //
58 // Member Data:
59 //
60 // fRmin Inside radius
61 // fRmax Outside radius
62 // fRtor swept radius of torus
63 //
64 // fSPhi The starting phi angle in radians,
65 // adjusted such that fSPhi+fDPhi<=2PI, fSPhi>-2PI
66 //
67 // fDPhi Delta angle of the segment in radians
68 //
69 // You could find very often in G4Torus functions values like 'pt' or
70 // 'it'. These are the distances from p or i G4ThreeVector points in the
71 // plane (Z axis points p or i) to fRtor point in XY plane. This value is
72 // similar to rho for G4Tubs and is used for definiton of the point
73 // relative to fRmin and fRmax, i.e. for solution of inside/outside
74 // problems
