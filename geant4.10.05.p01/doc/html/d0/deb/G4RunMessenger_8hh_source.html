31 // class description:
32 //
33 // This is a messenger class for G4RunManager.
34 // Implemented commands are following;
35 //
36 // Commands :
37 // beamOn * Start a Run.
38 // verbose * Set the Verbose level of G4RunManager.
39 // printProgress * Set the frequency of printing out the progress of a run.
40 // dumpRegion * Dump information of a region.
41 // dumpCouples * Dump information of material-cuts-couples.
42 // optimizeGeometry * Set the optimization flag of closing geometry.
43 // breakAtBeginOfEvent * Set a break point at the beginning of every event.
44 // breakAtEndOfEvent * Set a break point at the end of every event.
45 // abort * Abort current run processing.
46 // Initialize * Initialize G4 kernel.
47 // geometryModified * Force geometry to be closed again.
48 // physicsModified * Force cross-section tables to be calculated again.
49 // (and rebuilding physics table will be invoked)
50 // constructScoringWorlds * Construct scoring world(s) if defined
51 //
