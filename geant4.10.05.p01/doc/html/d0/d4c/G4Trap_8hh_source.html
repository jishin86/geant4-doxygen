32 // G4Trap
33 //
34 // Class description:
35 //
36 // A G4Trap is a general trapezoid: The faces perpendicular to the
37 // z planes are trapezia, and their centres are not necessarily on
38 // a line parallel to the z axis.
39 //
40 // Note that of the 11 parameters described below, only 9 are really
41 // independent - a check for planarity is made in the calculation of the
42 // equation for each plane. If the planes are not parallel, a call to
43 // G4Exception is made.
44 //
45 // pDz Half-length along the z-axis
46 // pTheta Polar angle of the line joining the centres of the faces
47 // at -/+pDz
48 // pPhi Azimuthal angle of the line joing the centre of the face at
49 // -pDz to the centre of the face at +pDz
50 // pDy1 Half-length along y of the face at -pDz
51 // pDx1 Half-length along x of the side at y=-pDy1 of the face at -pDz
52 // pDx2 Half-length along x of the side at y=+pDy1 of the face at -pDz
53 // pAlp1 Angle with respect to the y axis from the centre of the side
54 // at y=-pDy1 to the centre at y=+pDy1 of the face at -pDz
55 //
56 // pDy2 Half-length along y of the face at +pDz
57 // pDx3 Half-length along x of the side at y=-pDy2 of the face at +pDz
58 // pDx4 Half-length along x of the side at y=+pDy2 of the face at +pDz
59 // pAlp2 Angle with respect to the y axis from the centre of the side
60 // at y=-pDy2 to the centre at y=+pDy2 of the face at +pDz
61 //
62 //
63 // Member Data:
64 //
65 // fDz Half-length along the z axis
66 // fTthetaCphi = std::tan(pTheta)*std::cos(pPhi)
67 // fTthetaSphi = std::tan(pTheta)*std::sin(pPhi)
68 // These combinations are suitable for creation of the trapezoid corners
69 //
70 // fDy1 Half-length along y of the face at -fDz
71 // fDx1 Half-length along x of the side at y=-fDy1 of the face at -fDz
72 // fDx2 Half-length along x of the side at y=+fDy1 of the face at -fDz
73 // fTalpha1 Tan of Angle with respect to the y axis from the centre of
74 // the side at y=-fDy1 to the centre at y=+fDy1 of the face
75 // at -fDz
76 //
77 // fDy2 Half-length along y of the face at +fDz
78 // fDx3 Half-length along x of the side at y=-fDy2 of the face at +fDz
79 // fDx4 Half-length along x of the side at y=+fDy2 of the face at +fDz
80 // fTalpha2 Tan of Angle with respect to the y axis from the centre of
81 // the side at y=-fDy2 to the centre at y=+fDy2 of the face
82 // at +fDz
83 //
84 // TrapSidePlane fPlanes[4] Plane equations of the faces not at +/-fDz
85 // NOTE: order is important !!!
