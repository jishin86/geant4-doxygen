30 // Class Description:
31 //
32 // G4VUserVisAction is added to the scene by the command
33 // /vis/scene/add/userAction, which instantiates a G4CallbackModel to
34 // add to the RunDurationModels. Thus the pure virtual Draw() method
35 // is invoked whenever the viewer needs to "visit the kernel", e.g.,
36 // to remake its graphical database, if any, or simply to refresh the
37 // screen. operator() is defined to satisfy the template
38 // G4CallbackModel<G4VUserVisAction>.
39 //
40 // See the User Guide for Application Developers, Section 8.8.7 and 8.8.8.
