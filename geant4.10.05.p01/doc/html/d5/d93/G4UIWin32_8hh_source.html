43 // Class description :
44 //
45 // G4UIWin32 : class to handle a Windows interactive session.
46 // G4UIWin32 is the Windows version of G4UIterminal.
47 //
48 // A command box is at disposal for entering/recalling Geant4 commands.
49 // A menubar could be customized through the AddMenu, AddButton methods.
50 // Note that there are corresponding Geant4 commands to add a
51 // menus in the menubar and add buttons in a menu.
52 // Ex :
53 // /gui/addMenu test Test
54 // /gui/addButton test Init /run/initialize
55 // /gui/addButton test "Set gun" "/control/execute gun.g4m"
56 // /gui/addButton test "Run one event" "/run/beamOn 1"
57 //
58 // Command completion, by typing "tab" key, is available on the
59 // command line.
60 //
61 // Class description - end :
