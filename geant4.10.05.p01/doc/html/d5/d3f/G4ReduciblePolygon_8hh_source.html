33 // G4ReduciblePolygon.hh
34 //
35 // Class description:
36 //
37 // Utility class used to specify, test, reduce, and/or otherwise
38 // manipulate a 2D polygon.
39 //
40 // For this class, a polygon consists of n > 2 points in 2D
41 // space (a,b). The polygon is always closed by connecting the
42 // last point to the first. A G4ReduciblePolygon is guaranteed
43 // to fulfill this definition in all instances.
44 //
45 // Illegal manipulations (such that a valid polygon would be
46 // produced) result in an error return if possible and
47 // otherwise a G4Exception.
48 //
49 // The set of manipulations is limited currently to what
50 // is needed for G4Polycone and G4Polyhedra.
51 
52 // Author:
53 // David C. Williams (davidw@scipp.ucsc.edu)
54 // --------------------------------------------------------------------
