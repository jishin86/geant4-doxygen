76  // Methods for adding raw GEANT4 objects to the scene handler. They
77  // must always be called in the triplet PreAddSolid, AddSolid and
78  // PostAddSolid. The transformation and visualization attributes
79  // must be set by the call to PreAddSolid. If your graphics system
80  // is sophisticated enough to handle a particular solid shape as a
81  // primitive, in your derived class write a function to override one
82  // or more of the following. See the implementation of
83  // G4VSceneHandler::AddSolid (const G4Box& box) for more
84  // suggestions. If not, please implement the base class invocation.
