32 // (Description)
33 // This is a primitive scorer class for scoring Surface Flux.
34 // Flux version assumes only for G4Sphere shape, and the surface
35 // is defined at the inside of the sphere.
36 // The current is given in the unit of area.
37 // e.g. (Number of tracks)/mm2.
38 //
39 // Surface is defined at the inside of sphere.
40 // Direction -Rmin +Rmax
41 // 0 IN || OUT ->|<- | fFlux_InOut
42 // 1 IN ->| | fFlux_In
43 // 2 OUT |<- | fFlux_Out
44 //
45 // Created: 2007-08-14 Tsukasa ASO
46 // 2010-07-22 Introduce Unit specification.
