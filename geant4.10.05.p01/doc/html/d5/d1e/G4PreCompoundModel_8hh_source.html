29 // Class Description
30 // Model implementation for pre-equilibrium decay models in geant4.
31 // To be used in your physics list, in case you neeed this kind of physics.
32 // Can be used as a stand-allone model, but also in conjunction with an intra-nuclear
33 // transport, or any of the string-parton models.
34 // Class Description - End
35 //
36 // Modified:
37 // 03.09.2008 J.M.Quesada added external choice of inverse
38 // cross section option.(default OPTxs=3)
39 // 06.09.2008 J.M.Quesada external choices have been added for:
40 // - superimposed Coulomb barrier (if useSICB=true, default false)
41 // - "never go back" hipothesis (if useNGB=true, default false)
42 // - soft cutoff from preeq. to equlibrium (if useSCO=true, default false)
43 // - CEM transition probabilities (if useCEMtr=true)
