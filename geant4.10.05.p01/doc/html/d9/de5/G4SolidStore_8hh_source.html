28 // class G4SolidStore
29 //
30 // Class description:
31 //
32 // Container for all solids, with functionality derived from
33 // std::vector<T>. The class is a `singleton', in that only
34 // one can exist, and access is provided via the static method
35 // G4SolidStore::GetInstance().
36 //
37 // All solids should be registered with G4SolidStore, and removed on their
38 // destruction. Intended principally for UI browser. The underlying
39 // container initially has a capacity of 100.
40 //
41 // If much additional functionality is added, should consider containment
42 // instead of inheritance for std::vector<T>
43 //
