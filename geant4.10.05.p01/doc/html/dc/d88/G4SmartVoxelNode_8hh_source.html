28 // class G4SmartVoxelNode
29 //
30 // Class description:
31 //
32 // A node in the smart voxel hierarchy - a `slice' of space along a given
33 // axis between given minima and maxima. Note that the node is not aware
34 // of its position - this information being available/derivable by the
35 // node's owner(s) (voxelheaders).
36 //
37 // Member Data:
38 //
