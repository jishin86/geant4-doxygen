30 // Class G4TessellatedGeometryAlgorithms
31 //
32 // Class description:
33 //
34 // The G4TessellatedGeometryAlgorithms class is used to contain standard
35 // routines to determine whether (and if so where) simple geometric shapes
36 // intersect.
37 //
38 // The constructor doesn't need to do anything, and neither does the
39 // destructor.
40 //
41 // IntersectLineAndTriangle2D
42 // Determines whether there is an intersection between a line defined
43 // by r = p + s.v and a triangle defined by verticies P0, P0+E0 and P0+E1.
44 // Here:
45 // p = 2D vector
46 // s = scaler on [0,infinity)
47 // v = 2D vector
48 // P0, E0 and E1 are 2D vectors
49 // Information about where the intersection occurs is returned in the
50 // variable location.
51 //
52 // IntersectLineAndLineSegment2D
53 // Determines whether there is an intersection between a line defined
54 // by r = P0 + s.D0 and a line-segment with endpoints P1 and P1+D1.
55 // Here:
56 // P0 = 2D vector
57 // s = scaler on [0,infinity)
58 // D0 = 2D vector
59 // P1 and D1 are 2D vectors
60 // Information about where the intersection occurs is returned in the
61 // variable location.
