32 // G4Para
33 //
34 // Class description:
35 //
36 // A G4Parallepiped, essentially a box with half lengths dx,dy,dz
37 // `skewed' so that there are angles theta & phi of the polar line
38 // joining the faces at +-dz in z, and alpha formed by the y axis
39 // and the plane joinng the centre of the faces G4Parallel to the
40 // z-x plane at -dy and +dy.
41 //
42 // A G4Para is defined by:
43 // dx,dy,dz - Half-length in x,y,z
44 // alpha - Angle formed by the y axis and by the plane joining
45 // the centre of the faces G4Parallel to the z-x plane
46 // at -dy and +dy
47 // theta - Polar angle of the line joining the centres of the
48 // faces at -dz and +dz in z
49 // phi - Azimuthal angle of the line joining the centres of the
50 // faces at -dz and +dz in z
51 // Member data:
52 //
53 // Note that the angles parameters are not stored - precomputed trig is
54 // stored instead.
55 //
56 // fDx Half-length in x
57 // fDy Half-length in y
58 // fDz Half-length in z
59 //
60 // fTalpha Tan of alpha
61 // fTthetaCphi Tan theta * Cos phi
62 // fTthetaSphi Tan theta * Sin phi
63
