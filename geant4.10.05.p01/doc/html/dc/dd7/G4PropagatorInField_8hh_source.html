29 // Class G4PropagatorInField
30 //
31 // class description:
32 //
33 // This class performs the navigation/propagation of a particle/track
34 // in a magnetic field. The field is in general non-uniform.
35 // For the calculation of the path, it relies on the class G4ChordFinder.
36 //
37 // Key Method: ComputeStep(..)
