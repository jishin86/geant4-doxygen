29 // Class G4LogicalSurface
31 //
32 // Class description:
33 //
34 // An abstraction of a geometrical surface, it is an abstract
35 // base class for different implementations of surfaces.
36 // Its primary function is to hold pointers to objects that describe the
37 // surface's physical properties. For example it holds a pointer to a
38 // surface's optical properties, and because of this it is used in processes
39 // like G4OpBoundaryProcess.
