28 // class G4TransportationManager
29 //
30 // Class description:
31 //
32 // A singleton class which stores the (volume) navigator used by
33 // the transportation process to do the geometrical tracking.
34 // It also stores a pointer to the propagator used in a (magnetic)
35 // field and to the field manager.
36 // The class instance is created before main() is called, and
37 // in turn creates the navigator and the rest.
