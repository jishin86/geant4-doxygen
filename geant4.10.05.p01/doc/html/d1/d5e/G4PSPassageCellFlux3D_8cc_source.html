32 // (Description)
33 // This is a primitive scorer class for scoring cell flux.
34 // The Cell Flux is defined by a track length divided by a geometry
35 // volume, where only tracks passing through the geometry are taken
36 // into account. e.g. the unit of Cell Flux is mm/mm3.
37 //
38 // If you want to score all tracks in the geometry volume,
39 // please use G4PSCellFlux.
40 //
41 // Created: 2007-08-14 Tsukasa ASO
