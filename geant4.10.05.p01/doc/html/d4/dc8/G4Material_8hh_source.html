30 //
31 // ClassName: G4Material
32 //
33 // Description: Contains material properties
34 //
35 // Class description:
36 //
37 // Is used to define the material composition of Geant4 volumes.
38 // A G4Material is always made of G4Elements. It should has the name,
39 // the list of G4Elements, material density, material state, temperature,
40 // pressure. Other parameters are optional and may be set by the user code
41 // or computed at initialisation.
42 //
43 // There is several ways to construct G4Material:
44 // - from single element;
45 // - from a list of components (elements or other materials);
46 // - from internal Geant4 database of materials
47 //
48 // A collection of constituent Elements/Materials should be defined
49 // with specified weights by fractional mass or atom counts (only for Elements).
50 //
51 // Quantities, with physical meaning or not, which are constant in a given
52 // material are computed and stored here as Derived data members.
53 //
54 // The class contains as a private static member the Table of defined
55 // materials (an ordered vector of materials).
56 //
57 // It is strongly not recommended to delete materials in user code.
58 // All materials will be deleted automatically at the end of Geant4 session.
59 //
