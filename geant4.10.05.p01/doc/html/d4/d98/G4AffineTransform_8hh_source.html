29 // class G4AffineTransform
30 //
31 // Class description:
32 //
33 // A class for geometric affine transformations [see, eg. Foley & Van Dam]
34 // Supports efficient arbitrary rotation & transformation of vectors and the
35 // computation of compound & inverse transformations. A `rotation flag' is
36 // maintained internally for greater computational efficiency for transforms
37 // that do not involve rotation.
38 //
39 // Interfaces to the CLHEP classes G4ThreeVector & G4RotationMatrix
40 //
41 // For member function descriptions, see comments by declarations. For
42 // additional clarification, also check the `const' declarations for
43 // functions & their parameters.
44 //
45 // Member data:
46 //
47 // G4double rxx,rxy,rxz;
48 // G4double ryx,ryy,ryz; A 3x3 rotation matrix - net rotation
49 // G4double rzx,rzy,rzz;
50 // G4double tx,ty,tz; Net translation
51 
52 // History:
53 // Paul R C Kent 6 Aug 1996 - initial version
54 //
55 // 19.09.1996 E.Tcherniaev:
56 // - direct access to the protected members of the G4RotationMatrix class
57 // replaced by access via public access functions
58 // - conversion of the rotation matrix to angle & axis used to get
59 // a possibility to remove "friend" from the G4RotationMatrix class
60 // 06.05.2018 E.Tcherniaev:
61 // - optimized InverseProduct
62 // - added methods for inverse transformation: InverseTrasformPoint,
63 // InverseTransformAxis, InverseNetRotation, InverseNetTranslation
64 // --------------------------------------------------------------------
