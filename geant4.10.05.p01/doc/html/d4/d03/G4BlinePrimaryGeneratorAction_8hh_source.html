34 // Class description:
35 //
36 // Defines the primary generator action used for tracing magnetic
37 // field lines.
38 // It generates the primary vertex by using the user defined primary
39 // generator action and replaces only the definition of the particle
40 // to be tracked by a "charged-geantino".
41 // The user can define start position for field line tracking in
42 // his/her own primary generator action.
