40 // class description:
41 //
42 // This is a class which stores hits collections generated at one event.
43 // This class is exclusively constructed by G4SDManager when the first
44 // hits collection of an event is passed to the manager, and this class
45 // object is deleted by G4RunManager when a G4Event class object is deleted.
46 // Almost all public methods must be used by Geant4 kernel classes and
47 // the user should not invoke them. The user can use two const methods,
48 // GetHC() and GetNumberOfCollections() for accessing to the stored hits
49 // collection(s).

90  {
91  G4int n = 0;
