29 // class G4CashKarpRKF45
30 //
31 // Class description:
32 //
33 // The Cash-Karp Runge-Kutta-Fehlberg 4/5 method is an embedded fourth
34 // order method (giving fifth-order accuracy) for the solution of an ODE.
35 // Two different fourth order estimates are calculated; their difference
36 // gives an error estimate. [ref. Numerical Recipes in C, 2nd Edition]
37 // It is used to integrate the equations of the motion of a particle
38 // in a magnetic field.
