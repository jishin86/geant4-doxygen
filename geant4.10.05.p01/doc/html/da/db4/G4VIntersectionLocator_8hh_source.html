29 // Class G4VIntersectionLocator
30 //
31 // class description:
32 //
33 // Base class for the calculation of the intersection point with a boundary
34 // when PropagationInField is used.
35 // Gives possibility to choose the method of intersection; concrete locators
36 // implemented are: G4SimpleLocator, G4MultiLevelLocator, G4BrentLocator.
37 //
38 // Key Method: EstimateIntersectionPoint()
