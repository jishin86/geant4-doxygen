28 // class G4SmartVoxelProxy
29 //
30 // Class description:
31 //
32 // Class for proxying smart voxels. The class represents either a header
33 // (in turn refering to more VoxelProxies) or a node. If created as a node,
34 // calls to GetHeader cause an exception, and likewise GetNode when a header.
35 //
36 // Note that the proxy does NOT gain deletion responsibility for proxied
37 // objects.
38 
39 // History:
40 // 12.07.95 P.Kent Initial version
41 // 03.08.95 P.Kent Updated to become non abstract class, removing
42 // HeaderProxy and NodeProxy derived classes
43 // --------------------------------------------------------------------
