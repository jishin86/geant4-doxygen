32 // G4Cons
33 //
34 // Class description:
35 //
36 // A G4Cons is, in the general case, a Phi segment of a cone, with
37 // half-length fDz, inner and outer radii specified at -fDz and +fDz.
38 // The Phi segment is described by a starting fSPhi angle, and the
39 // +fDPhi delta angle for the shape.
40 // If the delta angle is >=2*pi, the shape is treated as continuous
41 // in Phi
42 //
43 // Member Data:
44 //
45 // fRmin1 inside radius at -fDz
46 // fRmin2 inside radius at +fDz
47 // fRmax1 outside radius at -fDz
48 // fRmax2 outside radius at +fDz
49 // fDz half length in z
50 //
51 // fSPhi starting angle of the segment in radians
52 // fDPhi delta angle of the segment in radians
53 //
54 // fPhiFullCone Boolean variable used for indicate the Phi Section
55 //
56 // Note:
57 // Internally fSPhi & fDPhi are adjusted so that fDPhi<=2PI,
58 // and fDPhi+fSPhi<=2PI. This enables simpler comparisons to be
59 // made with (say) Phi of a point.
60
